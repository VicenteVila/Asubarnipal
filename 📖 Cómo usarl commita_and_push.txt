📖 Cómo usarlo
Paso 1: Verificar que existe (opcional)
Abre el explorador de archivos de Windows y navega a C:\Asubarnipal. Deberías ver el archivo commit_and_push.bat.
Paso 2: Ejecutar
Doble clic sobre el archivo commit_and_push.bat
Se abrirá una ventana de comandos (negra) que mostrará:
================================================
   ASUBARNIPAL - GIT COMMIT & PUSH
================================================
[1/4] Verificando estado de git...
...
⏰ Cuando usarlo
Momento	Qué hacer
Ahora	Para subir los cambios que hicimos hoy
Después de modificar código	Cada vez que hagas cambios y quieras subirlos
Antes de reinstalar/restablecer	Para asegurar que tienes todo en GitHub
Semanalmente	Buena práctica de backup
🔄 Flujo recomendado
1. Haces cambios en el código (desde WSL o editor)
2. Cuando terminas, ejecutas commit_and_push.bat
3. Se crea commit + push automático
4. Listo!
